# Dionisos Contracts

This package uses **Foundry** and targets **Base (EVM)**.

## Install Foundry

```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

## Setup dependencies (OpenZeppelin + forge-std)

```bash
pnpm -C packages/contracts setup
```

## Run tests

```bash
pnpm -C packages/contracts test
```

## Build

```bash
pnpm -C packages/contracts build
```
